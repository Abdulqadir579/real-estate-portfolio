## Run Locally

Clone the project

```bash
  git clone https://github.com/IHARNAZAROV/real-estate-portfolio
```

Go to the project directory

```bash
  cd real-estate-portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Deployment

To deploy this project run

```bash
  npm run deploy
```
